Estructura Semántica HTML5

Esta es una representación visual de la estructura semántica recomendada para documentos HTML5 modernos.

!DOCTYPE html
html
head

(metadatos, título, enlaces a CSS/JS)

body
header

(logo, menú de navegación principal)

nav

(enlaces de navegación)

main

(contenido principal único en la página)

section

(agrupación temática de contenido)

article

(contenido independiente y autónomo)

aside

(contenido relacionado indirectamente)

footer

(información del pie de página)

Código de la estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la Página</title>
    <link rel="stylesheet" href="estilos.css">
</head>
<body>
    <header>
        <h1>Encabezado de la Página</h1>
        <nav>
            <!-- Menú de navegación -->
        </nav>
    </header>
    
    <main>
        <section>
            <h2>Sección Principal</h2>
            <article>
                <h3>Artículo 1</h3>
                <p>Contenido del artículo...</p>
            </article>
        </section>
        
        <aside>
            <h3>Contenido Relacionado</h3>
            <!-- Contenido relacionado -->
        </aside>
    </main>
    
    <footer>
        <p>Derechos reservados © 2023</p>
    </footer>
    
    <script src="script.js"></script>
</body>
</html>

¿Por qué usar elementos semánticos?

  • Mejoran la accesibilidad para usuarios con discapacidades
  • Facilitan el posicionamiento SEO
  • Hacen el código más legible y mantenible
  • Proporcionan una estructura clara a los motores de búsqueda
